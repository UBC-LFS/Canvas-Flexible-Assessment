Assessments

Click the +Assessment button to add all of the assessments in your course. If you have several assessments within a category, enter the category name (e.g. Quizzes, Discussions) unless you want to make the individual assessments within the category flexible. The Default % will be used for students who do not enter desired percentages. The Minimum % and Maximum % values define the range of weights students will be able to enter. If you do not want an assessment to be flexible, enter the same value for the Default %, Minimum % and Maximum %. The default values for all assessments must add up to 100%.
